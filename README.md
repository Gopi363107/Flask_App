    <h1>🔥 Forest Fire Prediction - Flask ML App</h1>

    <p>This is a web application built with <strong>Flask</strong> and <strong>Machine Learning</strong> that predicts the chances of forest fires using environmental data such as temperature, humidity, wind speed, and rainfall.</p>

    <h2>🌐 Live Demo</h2>
    <p>🔗 <a href="https://forest-fire-prediction-jnk2.onrender.com" target="_blank">https://forest-fire-prediction-jnk2.onrender.com</a></p>

    <h2>🚀 Features</h2>
    <ul>
      <li>Flask web framework</li>
      <li>Machine learning models: Lasso, Ridge, ElasticNet</li>
      <li>Cross-validation using LassoCV, RidgeCV, ElasticNetCV</li>
      <li>Clean and responsive frontend (HTML/CSS)</li>
      <li>Deployed for free on Render</li>
    </ul>

    <h2>🛠 Technologies Used</h2>
    <ul>
      <li>Python</li>
      <li>Flask</li>
      <li>Scikit-learn</li>
      <li>HTML5 + CSS3</li>
      <li>Render (deployment)</li>
    </ul>

    <h2>📁 Dataset</h2>
    <p>Used the <strong>Algerian Forest Fires Dataset</strong> for training and testing.</p>
